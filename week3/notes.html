Week 3 Notes: Objects | DOM | Events

Chapter 5: Objects

In JavaScript, Objects can be key-value pairs inside {}(curly brackets) called a property, can be used as parameters to functions, are variables but they can contain many values.

  • Object Properties Methods
    Car car.name: Chevrolet car.start()
    car.model: LT1 car.drive()
    car.weight: 1547kg car.brake()
    car.color: Black car.stop()

  • All cars have the same properties, but the property values differ from car to car.

  • All cars have the same methods, but the methods are performed at different times.

    • This always refers to the object it belongs to. This itself is not given a value. It's value depends on where it is placed. Here is a code i found from W3 Schools:

    var person = {
    ---firstName: "John",
    ---lastName : "Doe",
    ---id : 5566,
    ---fullName : function() {
    ------return this.firstName + " " + this.lastName;
    ---}
    };

    Because This is directly after the person object, that object is the owner of the function that uses This.

    Chapter 6: Documents Object Model - DOM

    The Document Object Model, or DOM for short, represents an HTML document as a network of connected nodes that form a tree-like structure, is a standard object model and programming interface for HTML.

  • document.getElementById(), document.getElementsByClassName(), document.getElementsByTagNames(), document.querySelector() can be used to access elements on a page.

  • parentNode(), previousSibling(), nextSibling(), childNodes(), children() methods can be used to navigate around the DOM tree.

  • An elementâ€™s attributes can be accessed using the getAttribute() method, and updated using the setAttribute() method.

  • The createElement() and createTextNode() methods can be used to create dynamic markup on the fly.

  • Markup can be added to the page using the appendChild() and insertBefore() methods.

  • Elements can be replaced using the replaceChild() method, and removed using the removeChild() method.

  • The innerHTML property can be used to insert raw HTML directly into the DOM.

  • CSS - getComputedStyle()

    • Chapter 7: Events

  • There are many types of event, including mouse events, keyboard events, and touch events.

  • We can remove an event using the removeEventListener method.

  • The default behavior of elements can be prevented using the preventDefault() function.

  • Event propagation is the order the events fire on each element.

  • Event delegation is when an event listener is added to a parent element to capture events that happen to its children elements.
