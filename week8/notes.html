Week 8 Notes: HTML5 and CSS3 for the Real world

  • CSS3 Transforms and Transitions

      • Transforms move or change the appearance of an element, while transitions make the element smoothly and gradually change from one state to another.

        The following properties:
          Transforms
        • Scaling
        • Rotation
        • Skew

          Transition
        • transition-property
        • transition-duration
        • transition-timing-function
        • transition-delay

    • SVG

      SVG stands for Scalable Vector Graphics. A specific file format that allows you to describe vector graphics using XML(eXtensible Markup Language: a system meant to annotate text. Just like us can use HTML tags to wrap our content and give it meaning, so can XML tags used to describe data, such as the contents of a file.).

    • Drag and Drop API

      This API allows us to specify that certain elements are draggable, and then specify what should happen when these draggable elements are dragged over or dropped onto other elements on the page.
